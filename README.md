# Crime Data Analysis using Machine Learning

## 📌 Overview

This project analyzes historical crime data to identify patterns and predict trends using machine learning algorithms.

## 🚀 Features

* Data preprocessing and cleaning
* Crime trend visualization
* Machine learning models (Decision Tree, KNN, Naive Bayes, Random Forest)
* Accuracy comparison of models

## 🛠️ Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

## 📊 Output

* Crime trends by state and year
* High-risk area identification
* Model performance comparison

## ▶️ How to Run

1. Open the notebook in Google Colab
2. Upload dataset files
3. Run all cells

## 📎 Conclusion

This project demonstrates how machine learning can be used for real-world data analysis and prediction.
